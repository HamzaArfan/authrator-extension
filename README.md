# Authrator VS Code Extension

- Activity Bar view to manage Collections and APIs
- Guest mode (local storage) and Logged-in mode (server via proxy)
- Uses existing Authrator endpoints and brand colors

## Commands
- Authrator: Login
- Authrator: Sign Up
- Authrator: Logout
- Authrator: Refresh
- Authrator: Create Collection
- Authrator: Create API (from collection item context)

## Build
- npm install
- npm run compile
- Press F5 to launch Extension Host
- npm run package to build .vsix (requires `vsce` installed globally)
